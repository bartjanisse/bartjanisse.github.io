Retreives a handle for the specified vendor_id and product_is.

When the kernel is attached to this device it will try to detach it. After detaching it will claim the interface. If claiming is not succesfulle the application will be aborted.

Parameters
vendor_idThe macro VENDOR_ID specified in in controller.h and is used to retreive a handle to the Xbox controller device.
product_idThe macro PRODUCT_ID specified in in controller.h and is used to retreive a handle to the Xbox controller device.
Returns
0 if succefull.

A valid pointer to handle and button.

The structure buttons is filled with the current state.
