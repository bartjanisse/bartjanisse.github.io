This is the main function of the application. 

When this applicattion is called from a website this is the first function which will run. 

The website can call this application to get information about the connected Xbox 360 controllers or to controll the rumble actuator or leds. 

The call made from the website should have this format\-: {\bfseries wbcweb?$<$id$>$,$<$cmd$>$,$<$val$>$} 

The fields in here have the following meaning\-: \begin{TabularC}{2}
\hline
\rowcolor{lightgray}{\bf Field }&{\bf Meaning  }\\\cline{1-2}
xbcweb &this application  \\\cline{1-2}
id &controller id  \\\cline{1-2}
cmd &command  \\\cline{1-2}
val &value for the given command  \\\cline{1-2}
\end{TabularC}


The command can be one of the following\-: \begin{TabularC}{2}
\hline
\rowcolor{lightgray}{\bf Command }&{\bf Meaning  }\\\cline{1-2}
btns &Will return the button states of the controller. The id field is the controller id  \\\cline{1-2}
rumble &Controll the rumble actuator of the controller. The id field is the controller id, The val field is the speed  \\\cline{1-2}
led &ontroll the leds of the controller. The id field is the controller id, The val field is the \hyperlink{group__xbcdeamon_ga1e4b1d1d1b6272a280f18efef4db3f5e}{Leds} led pattern  \\\cline{1-2}
\end{TabularC}
