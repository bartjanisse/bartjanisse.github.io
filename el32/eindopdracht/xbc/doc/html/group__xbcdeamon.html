enum Leds

This enumeration contains all the command availlable to controll the LED's. The function setControllerLeds can be used to controll the states of the indvidual LED's.

Enumerator
blink_all 

All led's off

flash_1_on 

All blinking

flash_2_on

2 flashes, then on
